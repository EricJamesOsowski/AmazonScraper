# AmazonScraper

A simple web scraper using selenium that opens a new chrome browser, sorts by lowest price to highest, and selects the first 10 results (excluding the "Similar Results" section) and writes the URL for each to a text file, which is saved in a text file titled "AmazonResults" inside of your Chrome browsers default text browser.
To use the program, run it and follow the prompts in the console / terminal.
